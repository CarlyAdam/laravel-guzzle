# Laravel Guzzle Example

Laravel 5.7+Guzzle Example

### Installing
```
1-Git clone
2-Run command "Composer Install"
3-Define  "your api_url" in "App\Http\Controllers\ApiController"
5-Run command "php artisan serve"
```
